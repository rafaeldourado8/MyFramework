# MyFramework Video Player

React video player components with Mosaic support.

## Installation

```bash
npm install myframework-video-player
```

## Usage

```tsx
import { VideoPlayer, Mosaic } from 'myframework-video-player';

<VideoPlayer streamUrl="http://localhost:8080/stream" />
<Mosaic cameras={cameras} layout="2x2" />
```

See [documentation](https://github.com/rafaeldourado8/MyFramework/blob/master/docs/video-player.md) for details.
