# MyFramework Core

Core DDD framework with domain, application, infrastructure and app layers.

## Installation

```bash
pip install myframework-core
```

## Usage

```python
from myframework.core.domain import Entity, ValueObject, AggregateRoot
from myframework.core.application import UseCase, Result
from myframework.core.app import Application, Module
```

See [documentation](https://github.com/rafaeldourado8/MyFramework/blob/master/docs/core.md) for details.
